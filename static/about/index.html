Learn to code by solving puzzles





Fully customizable

You can make your own puzzles and curriculums. The game levels are scratch projects. Follow these steps:

  1. Clone this project and modify as you wish or start a new one


  2. When the puzzle is complete, set the _allGoalsAchieved Scratch variable to true


  3. Share your level in scratch.mit.edu


  4. Test it on this website by changing the url to use your project
